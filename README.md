📘 Linux Basics: Pipes, Commands & Shell Scripting
📌 Overview
This guide introduces essential Linux concepts including basic commands, pipes, and shell scripting. It is designed for beginners who want to work efficiently in a Linux environment and automate tasks.

🖥️ Basic Linux Commands
Here are some commonly used commands:

Command	Description
ls	List directory contents
cd	Change directory
pwd	Print current directory
cp	Copy files/directories
mv	Move or rename files
rm	Remove files/directories
mkdir	Create a directory
touch	Create an empty file
cat	Display file contents
echo	Print text to terminal
🔗 Pipes (|)
Pipes allow you to send the output of one command as input to another.

Syntax:
command1 | command2
Example:
ls -l | grep ".txt"
➡️ Lists files and filters only .txt files.

More Examples:
cat file.txt | sort
➡️ Sorts file contents

ps aux | grep python
➡️ Finds running Python processes

🔄 Redirection
Operator	Description
>	Redirect output (overwrite)
>>	Redirect output (append)
<	Input redirection
Example:
echo "Hello" > file.txt
📜 Shell Scripting Basics
A shell script is a file containing a series of commands.

1. Create a Script
touch script.sh
2. Add Shebang
#!/bin/bash
3. Example Script
#!/bin/bash

echo "Hello, World!"
4. Make Executable
chmod +x script.sh
5. Run Script
./script.sh
🔁 Variables
name="John"
echo "Hello $name"
🔀 Conditionals
if [ $name == "John" ]; then
  echo "Welcome John"
else
  echo "Unknown user"
fi
🔁 Loops
For Loop:
for i in 1 2 3
do
  echo $i
done
While Loop:
count=1
while [ $count -le 3 ]
do
  echo $count
  ((count++))
done
⚙️ Useful Commands in Scripts
Command	Use
grep	Search text

sed	Stream editing
cut	Extract columns
find	Search files
chmod	Change permissions
🚀 Example: Combining Everything
