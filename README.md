## 🖥️ Basic Commands

| Command | Description |
|--------|------------|
| `ls` | List files and directories |
| `cd` | Change directory |
| `pwd` | Show current directory |
| `cp` | Copy files |
| `mv` | Move/rename files |
| `rm` | Delete files |
| `mkdir` | Create directory |
| `touch` | Create file |
| `cat` | Display file content |

## 🔗 Pipes
